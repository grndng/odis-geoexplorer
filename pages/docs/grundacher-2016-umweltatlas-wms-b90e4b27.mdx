Gründächer 2016 (Umweltatlas)

Es werden mehrere Raumbezüge bereitgestellt:

* Block- und Blockteilflächen der Blockkarte 1 : 5.000 des Informationssystems Stadt und Umwelt (ISU5, 31.12.2015),
* Gebäudeflächen,
* Begrünte Dachteilflächen der Gebäude,
* Begrünte Dachteilflächen der nicht-überbauten Tiefgaragen.

Die folgende Tabelle beschreibt die Attribute der block- und blockteilflächenbezogenen Angaben (siehe auch [hier](Datenformatbeschreibung%5FGruendaecher2016.pdf)):

| **Feldname**  | **Beschreibung**                                                                                                                                                                                      |
| ------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| SCHLUESSEL    | Block- bzw. Blockteilflächenschlüssel der ISU5, Stand 31.12.2015                                                                                                                                      |
| TYP           | Flächentyp (Code) des Informationssystems Stadt und Umwelt, Stand 31.12.2015, siehe [Umweltatlaskarte 06.08](https://www.berlin.de/umweltatlas/nutzung/stadtstruktur/2015/karten/artikel.1000368.php) |
| TYPKLAR       | Flächentyp des Informationssystems Stadt und Umwelt, Stand 31.12.2015, siehe [Umweltatlaskarte 06.08](https://www.berlin.de/umweltatlas/nutzung/stadtstruktur/2015/karten/artikel.1000368.php)        |
| gruen\_m2     | begrünte Dachfläche, gesamt \[m²\]                                                                                                                                                                    |
| gint\_m2      | begrünte Dachfläche, intensiv \[m²\]                                                                                                                                                                  |
| gex\_m2       | begrünte Dachfläche, extensiv \[m²\]                                                                                                                                                                  |
| gruen\_p      | begrünte Dachfläche, gesamt bezogen auf die Gebäudegrundfläche des Blocks \[%\]                                                                                                                       |
| gint\_p       | begrünte Dachfläche, intensiv bezogen auf die Gebäudegrundfläche des Blocks \[%\]                                                                                                                     |
| gex\_p        | begrünte Dachfläche, extensiv bezogen auf die Gebäudegrundfläche des Blocks \[%\]                                                                                                                     |
| area\_geb     | Gebäudegrundfläche des Blocks \[m²\]                                                                                                                                                                  |
| anzahl\_gruen | Anzahl der Gebäude mit begrünter Dachfläche                                                                                                                                                           |
| anzahl\_geb   | Anzahl aller Gebäude                                                                                                                                                                                  |

Die folgende Tabelle beschreibt die Attribute der gebäudebezogenen Angaben:

| **Feldname** | **Beschreibung**                                                                                       |
| ------------ | ------------------------------------------------------------------------------------------------------ |
| geb\_id      | Gebäude-ID                                                                                             |
| geb\_nutz    | Gebäudenutzungsklasse                                                                                  |
| gruendach    | Gründach vorhanden / nicht vorhanden                                                                   |
| EX\_INT      | Begrünungskategorie; intensiv oder extensiv nach überwiegendem Flächenanteil von gint\_m2 oder gex\_m2 |
| tgruen\_m2   | begrünte Dachfläche, gesamt \[m²\]                                                                     |
| tgint\_m2    | begrünte Dachfläche, intensiv \[m²\]                                                                   |
| tgex\_m2     | begrünte Dachfläche, extensiv \[m²\]                                                                   |
| tgruen\_p    | begrünte Dachfläche, gesamt bezogen auf die Gebäudegrundfläche \[%\]                                   |
| tgint\_p     | begrünte Dachfläche, intensiv bezogen auf die Gebäudegrundfläche \[%\]                                 |
| tgex\_p      | begrünte Dachfläche, extensiv bezogen auf die Gebäudegrundfläche \[%\]                                 |
| tgeb\_area   | Gebäudegrundfläche \[m²\]                                                                              |
| nutz         | Herkunft der Gebäudedaten                                                                              |
