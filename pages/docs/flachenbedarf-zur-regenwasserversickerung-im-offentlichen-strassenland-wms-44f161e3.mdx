Flächenbedarf zur Regenwasserversickerung im öffentlichen Straßenland

Für das Thema liegen mehrere Ebenen vor:

* Straßenabschnitte zur Regenwasserversickerung,
* Straßenabschnitte zur Regenwasserversickerung, nur getrennte Versickerung,
* Straßenabschnitte mit Einschränkungen.

Die folgende Tabelle beschreibt die Attribute der Ebene Straßenabschnitte zur Regenwasserversickerung:

| **Feldname** | **Beschreibung**            |
| ------------ | --------------------------- |
| elem\_nr     | Detailnetz-Nr.              |
| bezirk       | Bezirk                      |
| strschl      | Straßennummer               |
| str\_name    | Straßenname                 |
| idealfläc    | Idealisierter Flächenbedarf |
| sachdatum    | Sachdatum                   |

Die folgende Tabelle beschreibt die Attribute der Ebene Straßenabschnitte zur Regenwasserversickerung, nur getrennte Versickerung. Die Versickerung soll aufgrund der hohen Verkehrsdichte für die Flächen des Autoverkehrs und des Rad-Fußverkehrs getrennt umgesetzt werden. Die ausgewiesene anteilige Fläche gilt jeweils für die genannten Verkehrsträger. In Absprache mit der Wasserbehörde ist die Entwässerung der Fläche des Autoverkehrs ggf. als gedichtetes Mulden-Rigolen-System ausgeführt werden.

| **Feldname** | **Beschreibung**            |
| ------------ | --------------------------- |
| elem\_nr     | Detailnetz-Nr.              |
| bezirk       | Bezirk                      |
| strschl      | Straßennummer               |
| str\_name    | Straßenname                 |
| idealfläc    | Idealisierter Flächenbedarf |
| sachdatum    | Sachdatum                   |

Die folgende Tabelle beschreibt die Attribute der Ebene Straßenabschnitte mit Einschränkungen. Bei diesen Straßenabschnitten ist die Versickerung von Regenwasser nur mit Einschränkungen bzw. nicht möglich.
