Strat. Lärmkarte L\_DEN (Tag-Abend-Nacht-Index) Straßenverkehr 2017 sowie die Strat. Lärmkarte L\_N (Nacht-Index) Straßenverkehr 2017

Die Strategischen Lärmkarten 2017 geben Auskunft über die Lärmbelastung im Einwirkbereich von Hauptlärmquellen. Sie sind eine Fortführung der Strategischen Lärmkarten 2012.  
Die zip-Datei enthält zwei Geotiff-Dateien: die Strat. Lärmkarte L\_DEN (Tag-Abend-Nacht-Index) Straßenverkehr 2017 sowie die Strat. Lärmkarte L\_N (Nacht-Index) Straßenverkehr 2017.
