Flächennutzung, Stadtstruktur und Versiegelung 1990 (Umweltatlas)

Die folgende Tabelle zeigt die Attribute zur Flächennutzung, Stadtstruktur und Versiegelung 1990 (Umweltatlas).

| **Feldname** | **Beschreibung**                                                                                                                                                                        |
| ------------ | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| CODE         | Schlüssel                                                                                                                                                                               |
| BEZIRK       | Bezirk, alte Bezirksnummer, vor der Gebietsreform 2000                                                                                                                                  |
| STAGEB       | stat. Gebiet, Nummern der Stat. Gebiete des STALA (31.12.1990)                                                                                                                          |
| BLOCKNR      | Blocknummer, Nummern der Stat. Blöcke des STALA (31.12.1990)                                                                                                                            |
| TBLOCKNR     | Teilblocknummer, Unterteilung der statistischen Blöcke nach Art der Nutzung (Grün-Bebaut) und Bodengesellschaften (im Außenbereich)                                                     |
| NUTZ         | Nutzung, Flächennutzung (alle Felder der Codetabelle); bei auftretender Doppelnutzung (ca 750 Flächen) wird die Grün- und Freiflächennutzung (grz) angegeben                            |
| TYP          | Flächentyp, weitere Differenzierung der Flächennutzung, insbesondere bzgl. Stadtstruktur (Baualter, Siedlungsform) und der Gemeinbedarfsnutzungen, Code                                 |
| VG           | Versiegelung in %, Versiegelungsgrad der Bezugsfläche (bebaute und unbebaute Anteile, ohne Straßen) aus Satellitendaten und Luftbildern abgeleitet, Erstaufnahme für Gesamt-Berlin 1991 |

Ausführliche Informationen finden Sie in den Begleittexten der im Umweltatlas Berlin veröffentlichten Karte 06.01 (Ausgabe 1995).
