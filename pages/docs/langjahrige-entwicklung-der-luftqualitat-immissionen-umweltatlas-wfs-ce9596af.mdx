Entwicklung Luftqualität - Immissionen 

Darstellung aller Stationen und Messwerte der BLUME-, RUBIS- und Passivsammler-Messnetze seit 1975 sowie ausgewählter langjährig betriebener Berliner Klimastationen.

Die folgende Tabelle beschreibt die Attribute:

| **Feldname** | **Beschreibung**                                                                      |
| ------------ | ------------------------------------------------------------------------------------- |
| id           | Stationsname                                                                          |
| link         | Verlinkung zu HTML-Seiten mit Downloadmöglichkeiten tabellarischer Zusammenstellungen |
| messnetz     | Bezeichnung des Messnetzes                                                            |
| stand        | Betriebsstatus des Messnetzes                                                         |
| stattyp      | Belastungsregime des Messnetzes                                                       |
| plz          | Postleitzahl des Messstandortes                                                       |
| adresse      | Straße und Hausnummer des Messstandortes                                              |
