Starkregenhinweiskarte (Umweltatlas)

Es werden mehrere Kartenebenen in unterschiedlichen Raumeinheiten dargestellt:

* Topographische Senkenanalyse,
* Starkregengefahrenkarte vorhanden,
* Feuerwehreinsätze (Block(teil)flächen ISU5),
* Feuerwehreinsätze (Punkte).

Die folgende Tabelle beschreibt die Attribute der Ebene Topographische Senkenanalyse:

| **Feldname** | **Beschreibung**                |
| ------------ | ------------------------------- |
| gisid        | ID                              |
| drainarea    | Fläche Einzugsgebiet \[m²\]     |
| fillarea     | Fläche Senke \[m²\]             |
| filldepth    | Maximale Tiefe der Senke \[cm\] |
| bottomelev   | Geländehöhe Senkenbasis \[m\]   |
| fillvolume   | Füllvolumen \[m³\]              |

Die folgende Tabelle beschreibt die Attribute der Ebene Starkregengefahrenkarte vorhanden:

| **Feldname** | **Beschreibung**                                                               |
| ------------ | ------------------------------------------------------------------------------ |
| gisid        | ID                                                                             |
| name         | Gebietsname, Gebiet für das eine detaillierte 1D/2D-Simulationen vorhanden ist |

Die folgende Tabelle beschreibt die Attribute der Ebene Feuerwehreinsätze (Block(teil)flächen ISU5):

| **Feldname** | **Beschreibung**                          |
| ------------ | ----------------------------------------- |
| gisid        | ID                                        |
| count        | Einsatzanzahl je Blockteil-/Straßenfläche |

Die folgende Tabelle beschreibt die Attribute der Ebene Feuerwehreinsätze (Punkte):

| **Feldname** | **Beschreibung**                      |
| ------------ | ------------------------------------- |
| gisid        | ID                                    |
| anzahl       | Anzahl der Einsätze                   |
| angelegt     | Datum (angelegt)                      |
| t            | anzunehmendes Wiederkehrintervall (T) |
| ortsteil     | Ortsteil                              |
