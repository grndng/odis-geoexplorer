Strat. Lärmkarte Gesamtlärmindex L\_DEN (Tag-Abend-Nacht-Index) Raster 2017 sowie die Strat. Lärmkarte Gesamtlärmindex L\_N (Nacht-Index) Raster 2017

Die Karte stellt über die Anforderungen der Umgebungslärmrichtlinie hinaus eine summarische Betrachtung (Pegeladdition) der untersuchten Verkehrslärmquellen dar. Sie führt die entsprechende Themenkarte aus dem Jahre 2012 fort.  
Die zip-Datei enthält zwei Geotiff-Dateien: die Strat. Lärmkarte Gesamtlärmindex L\_DEN (Tag-Abend-Nacht-Index) Raster 2017 sowie die Strat. Lärmkarte Gesamtlärmindex L\_N (Nacht-Index) Raster 2017\. Die Daten liegen erstmals im Koordinatensystem ETRS89 / UTM zone 33N (EPSG-Code: 25833) vor.
