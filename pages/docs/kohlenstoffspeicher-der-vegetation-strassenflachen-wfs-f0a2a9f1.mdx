Kohlenstoffspeicher der Vegetation

Zum Thema **Kohlenstoffspeicher der Vegetation**  werden zwei Geometrien abgegeben.

| **Feldname**   | **Beschreibung**                                                                                                                          |

| area\_unver    | unversiegelte Fläche \[m²\]                                                                                                               |

| woz\_name      | Nutzung der bebauten Flächen                                                                                                              |
| grz\_name      | Grün- und Freiflächennutzung                                                                                                              |
| ststrname      | Stadtstrukturtyp                                                                                                                          |
| typklar        | Flächentyp                                                                                                                                |
| natkosntyp     | NatKos-Nutzungstyp (codiert)  (?: sonstige/ohne Nutzung, KG: Kleingarten, S: Siedlung, W: Forst, VS: Straße, P: Park, LW: Landwirtschaft) |
