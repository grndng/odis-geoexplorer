07.05 Strat. Lärmkarten Fassadenpegel 2017

Der Datensatz enthält eine txt-Datei mit allen Fassadenpunkten der Berliner Wohngebäude (Koordinaten ETRS89 / UTM Zone 33N, EPSG: 25833, Gesamtlärm, Straßenverkehr, Straßenbahn und oberird. U-Bahn, Flugverkehr Flughafen Tegel, Flugverkehr Flughafen Schönefeld).

| Feldname    | Beschreibung                                                                    |
| ----------- | ------------------------------------------------------------------------------- |
| XKOORD      | X-Koordinate ETRS89 / UTM Zone 33N, EPSG: 25833                                 |
| YKOORD      | Y-Koordinate ETRS89 / UTM Zone 33N, EPSG: 25833                                 |
| L\_STR\_DEN | Lärmindex L\_DEN (Tag/Abend/Nacht) des Straßenverkehrs in dB(A)                 |
| L\_STR\_N   | Lärmindex L\_N (Nacht) des Straßenverkehrs in dB(A)                             |
| L\_SCH\_DEN | Lärmindex L\_DEN (Tag/Abend/Nacht) von Straßenbahn und oberird. U-Bahn in dB(A) |
| L\_SCH\_N   | Lärmindex L\_N (Nacht) von Straßenbahn und oberird. U-Bahn in dB(A)             |
| L\_TXL\_DEN | Lärmindex L\_DEN (Tag/Abend/Nacht) Flugverkehr Flughafen Tegel in dB(A)         |
| L\_TXL\_N   | Lärmindex L\_N (Nacht) Flugverkehr Flughafen Tegel in dB(A)                     |
| L\_SXF\_DEN | Lärmindex L\_DEN (Tag/Abend/Nacht) Flugverkehr Flughafen Schönefeld in dB(A)    |
