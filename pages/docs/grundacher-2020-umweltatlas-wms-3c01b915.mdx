Gründächer 2020 (Umweltatlas)

Es werden mehrere Raumbezüge bereitgestellt:

* Block- und Blockteilflächen der Blockkarte 1 : 5.000 des Informationssystems Stadt und Umwelt (ISU5, 31.12.2020),
* Gebäudeflächen,
* Begrünte Dachteilflächen der Gebäude,
* Begrünte Dachteilflächen der nicht-überbauten Tiefgaragen.

Die folgende Tabelle beschreibt die Attribute der block- und blockteilflächenbezogenen Angaben:

| **Feldname** | **Beschreibung**                                                                                                                                                                                      |
| ------------ | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| schluessel   | Block- bzw. Blockteilflächenschlüssel der ISU5, Stand 31.12.2020                                                                                                                                      |
| typ          | Flächentyp (Code) des Informationssystems Stadt und Umwelt, Stand 31.12.2020, siehe [Umweltatlaskarte 06.08](https://www.berlin.de/umweltatlas/nutzung/stadtstruktur/2020/karten/artikel.1175314.php) |
| typklar      | Flächentyp des Informationssystems Stadt und Umwelt, Stand 31.12.2020, siehe [Umweltatlaskarte 06.08](https://www.berlin.de/umweltatlas/nutzung/stadtstruktur/2020/karten/artikel.1175314.php)        |
| gruen20\_m2  | begrünte Dachfläche, gesamt 2020 \[m²\]                                                                                                                                                               |
| gint20\_m2   | begrünte Dachfläche, intensiv 2020 \[m²\]                                                                                                                                                             |
| gex20\_m2    | begrünte Dachfläche, extensiv 2020 \[m²\]                                                                                                                                                             |
| gruen20\_p   | begrünte Dachfläche, gesamt bezogen auf die Gebäudegrundfläche des Blocks 2020 \[%\]                                                                                                                  |

| area\_geb    | Gebäudegrundfläche des Blocks 2020 \[m²\]                                                                                                                                                             |
| anzahl\_gru  | Anzahl der Gebäude mit begrünter Dachfläche 2020                                                                                                                                                      |
| anzahl\_geb  | Anzahl aller Gebäude 2020                                                                                                                                                                             |
| gruen16\_m2  | Begrünte Dachfläche 2016, gesamt \[m²\]                                                                                                                                                               |
