Strat. Lärmkarte L\_DEN (Tag-Abend-Nacht-Index) Flugverkehr 2012 und Strat. Lärmkarte L\_N (Nacht-Index) Flugverkehr 2012

Die Strategischen Lärmkarten 2012 geben Auskunft über die Lärmbelastung im Einwirkbereich von Hauptlärmquellen. Sie sind eine Fortführung der Strategischen Lärmkarten 2007.  
Die zip-Datei enthält zwei Geotiff-Dateien: die Strat. Lärmkarte L\_DEN (Tag-Abend-Nacht-Index) Flugverkehr 2012 sowie Strat. Lärmkarte L\_N (Nacht-Index) Flugverkehr 2012.
