Starkregengefahrenkarte (Umweltatlas)

Es werden mehrere Kartenebenen in unterschiedlichen Raumeinheiten dargestellt:

* Modellgebiete,
* Fließgeschwindigkeit,
* Wasserstand, verschiedene Ereignisse.

Die folgende Tabelle beschreibt die Attribute der Ebene Modellgebiete:

| **Feldname** | **Beschreibung**       |
| ------------ | ---------------------- |
| gisid        | ID                     |
| layer        | Name des Modellgebiets |
| jahr         | Erstellungsjahr        |

Die folgende Tabelle beschreibt die Attribute der Ebenen Fließgeschwindigkeit:

| **Feldname** | **Beschreibung**            |
| ------------ | --------------------------- |
| gisid        | ID                          |
| speed        | Fließgeschwindikeit \[m/s\] |
| angle\_deg   | Fließrichtung \[Grad\]      |

Die Ebenen Wasserstand, verschiedene Ereignisse werden als Atom-FEED als tiff-Dateien abgegeben.
